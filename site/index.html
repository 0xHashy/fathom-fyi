Quick Start

Running in under 2 minutes

Install Fathom Step 1 
# Install globally
$ npm install -g fathom-fyi

# Or run directly with npx
$ npx fathom-fyi
Claude Desktop — claude_desktop_config.json Step 2a 
{
  "mcpServers": {
    "fathom": {
      "command": "npx",
      "args": ["fathom-fyi"],
      "env": {
        "CG_API_KEY": "your_coingecko_key",
        "FRED_API_KEY": "your_fred_key",
        "API_TIER": "free"
      }
    }
  }
}
Cursor — .cursor/mcp.json Step 2b 
{
  "mcpServers": {
    "fathom": {
      "command": "npx",
      "args": ["fathom-fyi"],
      "env": {
        "CG_API_KEY": "your_coingecko_key",
        "FRED_API_KEY": "your_fred_key",
        "API_TIER": "free"
      }
    }
  }
}
Get your API keys (both free) Step 3 
# CoinGecko API key (free tier, 30 calls/min)
https://www.coingecko.com/en/api

# FRED API key (free, unlimited)
https://fred.stlouisfed.org/docs/api/api_key.html

# That's it. Both keys are free to obtain.
# Set API_TIER to "free" and you're live.
