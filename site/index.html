The Thesis March 2026

Every AI agent operating in financial markets today is flying blind.

Trading bots see prices. DeFi agents see yields. Treasury managers see balances. None of them understand context.

They cannot tell you if the market is in euphoria or capitulation. They do not know where we are in the Bitcoin halving cycle. They have no concept of whether macro conditions are a tailwind or headwind. They act on data without understanding.

Fathom closes this gap. It synthesizes five real-time data sources into interpreted intelligence — regime classification, cycle positioning, sentiment state, DeFi health, macro context, and on-chain signals.

Your agent does not receive "BTC is $82,000." It receives a complete financial reality briefing with risk scores, historical analogs, and a specific recommended posture. That is the difference between data and understanding.

Quick Start Configuration

Running in under two minutes

Install Step 1 
# Install globally
$ npm install -g fathom-fyi

# Or run directly
$ npx fathom-fyi
Claude Desktop — claude_desktop_config.json Step 2 
{
  "mcpServers": {
    "fathom": {
      "command": "npx",
      "args": ["fathom-fyi"],
      "env": {
        "CG_API_KEY": "your_coingecko_key",
        "FRED_API_KEY": "your_fred_key",
        "API_TIER": "free"
      }
    }
  }
}
Cursor — .cursor/mcp.json Step 2 alt 
{
  "mcpServers": {
    "fathom": {
      "command": "npx",
      "args": ["fathom-fyi"],
      "env": {
        "CG_API_KEY": "your_coingecko_key",
        "FRED_API_KEY": "your_fred_key",
        "API_TIER": "free"
      }
    }
  }
}
